Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent are, 1/innerHTML Gets or sets the HTML content within an element, including HTML tags, 2/innerText Gets or sets the visible text content of an element, as it appears on the rendered page, 3/Gets or sets the text content of an element, including all text from all descendant nodes, regardless of visibility.

Question-3: What is event delegation in the DOM?

Event delegation in JavaScript leverages the concept of event bubbling to efficiently handle events on multiple child elements by attaching a single event listener to a parent element.

Question-4: What is event bubbling in the DOM?

In JavaScript event bubbling a phase of event propagation where an event, after being triggered on a specific element, "bubbles up" through the DOM tree to its parent elements.
